⭐ 链地址法示意图（带冲突示例）

假设 HashMap 的数组容量是 8（index 从 0 到 7）：
HashMap bucket array (capacity = 8)

 index
   ↓
+------+------+------+------+------+------+------+------+
|  0   |  1   |  2   |  3   |  4   |  5   |  6   |  7   |
+------+------+------+------+------+------+------+------+
   |            |                     |
   |            |                     |
   v            v                     v
  null         Node A               Node C
                |                     |
                v                     v
              Node B                Node D

⭐ 示例：插入四个 key

假设插入以下 key：
key	  hash(key)	  index = hash % 8
"A"	    10	        10 % 8 = 2
"B"	    18	        18 % 8 = 2
"C"	    23	        23 % 8 = 7
"D"	    31	        31 % 8 = 7

你会发现：
"A" 和 "B" 落在 同一个桶 2
"C" 和 "D" 落在 同一个桶 7
因此形成两条链。

⭐ 更详细版本（显示 key/value/next）
index
  ↓
+------+------+------+------+------+------+------+------+
|  0   |  1   |  2   |  3   |  4   |  5   |  6   |  7   |
+------+------+------+------+------+------+------+------+
                |                                    |
                |                                    |
                v                                    v
          +----------------+                +----------------+
          | key:A value:10 |                | key:C value:30 |
          | next:   ------>|                | next:   ------>|
          +----------------+                +----------------+
                             |                         |
                             v                         v
                       +----------------+       +----------------+
                       | key:B value:20 |       | key:D value:40 |
                       | next:   null   |       | next:   null   |
                       +----------------+       +----------------+

说明：
index = 2 的桶指向 A，A.next 指向 B
index = 7 的桶指向 C，C.next 指向 D

⭐ 链地址法查找流程（get(key)）演示
例：get("B"):
    1. 算 index: "B" → hash 18 → 18 % 8 = 2
    2. 去 bucket[2]
    3. 遍历链表：
        - A？不是
        - B？是 → 返回 value 20
    查找时间取决于链表长度，但平均为 O(1)。

⭐ 链地址法插入流程（put(key, value)）演示
插入 "B" ：
    1. 计算 index = 2
    2. bucket[2] 里有 A
    3. 遍历链表
        - 找不到 B（说明 key 不存在）
    4. 插入链表头（HashMap 默认用头插法）：
    插入前：
        2 → A
    插入 "B"后（头插）：
        2 → B → A
    数组指向链表新的头节点。
